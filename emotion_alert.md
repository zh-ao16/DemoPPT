## 一、情绪预警概述

### 1.1 目的

建立用户情绪识别和预警机制，在用户表达负面情绪时及时介入，提升客户满意度，避免负面口碑扩散。

### 1.2 预警触发场景

- 用户提交工单时的描述
- 用户聊天/对话中的情绪表达
- 用户评价和反馈文字
- 连续操作失败的报错提示
- 长时间等待的操作反馈

## 三、情绪评分计算

### 3.1 计算公式

```
情绪得分 = Σ(关键词强度分值) × 情境系数

情境系数:
- 用户连续操作失败≥3次: ×1.5
- 用户等待时间>5分钟: ×1.3
- 用户是付费用户: ×1.2
- 用户提及"紧急/马上要用": ×1.5
```

### 3.2 预警等级划分

| 预警等级 | 得分范围 | 动作 |
|---------|---------|-----|
| 🔴 红色预警 | ≥15分 | 立即处理 + 客服主管 + 2小时内回访 |
| 🟠 橙色预警 | 8-14分 | 优先处理 + 24小时内回访 |
| 🟡 黄色预警 | 4-7分 | 正常处理 + 7天内回访 |
| 🟢 绿色 | 0-3分 | 正常流程 |

### 3.3 组合场景识别

| 场景组合 | 综合判定 | 动作 |
|---------|---------|-----|
| 退款+愤怒词 | 高退款风险 | 标记高风险+优先退款处理 |
| 崩溃+失望词 | 产品严重问题 | 标记P0+技术紧急修复 |
| 不会用+焦急 | 易用性问题 | 标记易用性优化+主动教学 |
| 逻辑不对+垃圾 | AI质量问题严重 | 标记AI优化+优先处理 |

## 四、预警响应机制

### 4.1 自动响应

| 预警等级 | 自动响应内容 |
|---------|-------------|
| 🔴 红色 | 立即发送安抚消息："非常抱歉给您带来不愉快的体验，我们已将您的问题标记为最高优先级，客服主管将直接跟进，预计2小时内与您联系。" |
| 🟠 橙色 | 发送优先处理消息："感谢您的反馈，我们已将您的问题提升处理优先级，预计24小时内为您解决。" |
| 🟡 黄色 | 发送感谢消息："感谢您的宝贵反馈，我们会认真评估并改进。" |

### 4.2 人工介入规则

| 触发条件 | 介入人员 | 响应时限 |
|---------|---------|---------|
| 红色预警 | 客服主管 + 技术负责人 | 30分钟内响应 |
| 橙色预警连续3次同一用户 | 客服主管 | 4小时内响应 |
| 退款+负面情绪 | 退款专项组 + 客服主管 | 2小时内响应 |
| 产品崩溃连续2次 | 技术负责人 | 1小时内响应 |

### 4.3 升级流程

```
黄色预警(4-7分)
    ↓ 48小时未解决
橙色预警(8-14分)
    ↓ 24小时未解决 或 得分≥15
红色预警(≥15分)
    ↓ 触发
客服主管直接介入 + 影响复盘
```

## 五、特殊场景情绪预警

### 5.1 PPT工具特定场景

#### 场景1：AI生成内容被否定
```
触发词：内容太水、废话、没用、全部重写、导师说、老板说
情绪强度：7-9分
特殊动作：标记为"AI质量优化"需求，优先转AI优化组
回访重点：询问具体哪些内容不达标，收集示例
```

#### 场景2：模板质量问题
```
触发词：模板丑、配色难看、太土、看起来廉价、不专业
情绪强度：5-7分
特殊动作：标记为"设计质量优化"需求，转设计组
回访重点：询问具体哪个模板问题，收集截图
```

#### 场景3：时间压力场景
```
触发词：马上要用、明天汇报、今天要交、着急、紧急
情绪强度：6-8分
特殊动作：标记"紧急"，优先处理+加急响应
回访重点：确认是否及时解决
```

#### 场景4：反复出现问题
```
触发词：（同一问题出现3次以上）
情绪强度：自动+3分
特殊动作：触发深度复盘+主动补偿评估
回访重点：道歉+补偿方案讨论
```

### 5.2 VIP用户情绪预警

| 用户类型 | 情绪预警阈值降低 | 响应时间压缩 |
|---------|----------------|-------------|
| 付费年度用户 | 得分≥5即触发红色 | 响应时间减半 |
| 企业用户 | 得分≥8即触发红色 | 响应时间减半 |
| 单次消费≥99元 | 得分≥6即触发红色 | 2小时内必须响应 |

## 六、情绪数据分析

### 6.1 周度情绪报告

| 指标 | 定义 |
|-----|------|
| 负面情绪占比 | 负面工单数/总工单数 |
| 平均情绪得分 | 所有工单情绪得分均值 |
| 高情绪工单数 | 红色预警工单数 |
| 情绪解决率 | 预警后用户满意度≥4的工单占比 |
| 情绪转化率 | 负面情绪用户转为正面评价的比例 |

### 6.2 月度复盘

- 情绪词云更新（每月新增高频负面词）
- 预警准确率评估
- 高频触发场景优化
- 情绪响应时效分析
