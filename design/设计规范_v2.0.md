## 三、间距系统

### 3.1 基础间距

| 名称 | 大小 | 用途 |
|-----|------|------|
| `--space-1` | 4px | 紧凑间距 |
| `--space-2` | 8px | 标签内间距 |
| `--space-3` | 12px | 小组件内间距 |
| `--space-4` | 16px | 标准间距 |
| `--space-5` | 20px | 较大间距 |
| `--space-6` | 24px | 区块间距 |
| `--space-8` | 32px | 大区块间距 |
| `--space-10` | 40px | 页面边距(移动) |
| `--space-12` | 48px | 页面边距(PC) |

### 3.2 布局间距

```css
.page-padding { padding: var(--space-12); }        /* PC端页面边距 */
.page-padding-mobile { padding: var(--space-4); }   /* 移动端页面边距 */
.section-gap { margin-bottom: var(--space-8); }     /* 区块垂直间距 */
.card-padding { padding: var(--space-6); }          /* 卡片内边距 */
.card-gap { gap: var(--space-4); }                  /* 卡片间距 */
.element-gap { margin-bottom: var(--space-4); }    /* 元素间距 */
```

## 六、组件规范

### 6.1 按钮

```css
.btn-primary {
  background: var(--color-brand);
  color: white;
  border: none;
  padding: 12px 24px;
  border-radius: var(--radius-md);
  font-size: 16px;
  cursor: pointer;
  transition: all 0.2s ease;
}

.btn-primary:hover {
  background: var(--color-brand-dark);
  transform: translateY(-2px);
}

.btn-secondary {
  background: white;
  color: var(--color-brand);
  border: 2px solid var(--color-brand);
  padding: 10px 22px;
  border-radius: var(--radius-md);
  cursor: pointer;
}
```

### 6.2 卡片

```css
.card {
  background: white;
  border-radius: var(--radius-lg);
  padding: var(--space-6);
  box-shadow: var(--shadow-md);
  transition: box-shadow 0.2s ease;
}

.card:hover {
  box-shadow: var(--shadow-lg);
}
```

### 6.3 输入框

```css
.input {
  border: 1px solid var(--color-border);
  border-radius: var(--radius-md);
  padding: 10px 16px;
  font-size: 16px;
  transition: border-color 0.2s;
}

.input:focus {
  outline: none;
  border-color: var(--color-brand);
}
```
