### 2.2 AI生成进度反馈组件（关键）

**设计目标**：让用户感知AI正在工作，消除等待焦虑

```
┌────────────────────────────────────────────────────────┐
│  ┌──────────────────────────────────────────────────┐  │
│  │  🔄 正在生成第 3/8 页...                           │  │
│  │  ████████████░░░░░░░░░░░░░░░  38%                │  │
│  │                                                  │  │
│  │  当前：大纲生成 → 内容填充 → 智能排版 → 配色优化  │  │
│  └──────────────────────────────────────────────────┘  │
└────────────────────────────────────────────────────────┘
```

**状态定义**：

| 状态 | 视觉表现 | 文案示例 |
|------|----------|----------|
| 空闲 | 品牌蓝按钮，文字"开始生成" | "开始生成" |
| 思考中 | 蓝底加载动画，步骤指示器激活 | "正在分析主题..." |
| 生成中 | 进度条+百分比+当前步骤 | "正在生成第 3/8 页..." |
| 完成 | 绿色成功提示，自动切换预览 | "生成完成！🎉" |
| 失败 | 红色错误提示，显示原因 | "生成失败，请重试" |

**进度条样式**：
- 轨道： `#E5E7EB`
- 填充：`var(--color-brand)` 渐变至 `var(--color-brand-dark)`
- 圆角：`full` (9999px)
- 高度：8px

**步骤指示器**：
- 4个步骤节点横排
- 当前步骤：品牌蓝填充 + 脉冲动画
- 已完成：品牌蓝 + 勾选图标
- 未开始：灰色边框

### 2.3 核心交互说明

1. **输入主题** → 自动触发防抖（500ms）预验证
2. **点击生成** → 按钮变为加载态，显示进度组件
3. **生成完成** → 自动滚动到预览区域，播放轻微入场动画
4. **生成失败** → 显示具体错误原因，提供"重试"和"联系客服"选项

## 七、交互规范汇总

### 7.1 动效规范

| 动效类型 | 时长 | 缓动函数 |
|----------|------|----------|
| 微交互 | 150ms | ease-out |
| 展开/收起 | 200ms | ease-in-out |
| 页面切换 | 300ms | ease-out |
| 模态框 | 250ms | cubic-bezier(0.4, 0, 0.2, 1) |
| 进度条 | 持续 | linear |

### 7.2 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|----------|
| Mobile | < 640px | 单列，底部导航 |
| Tablet | 640-1024px | 双栏，侧边导航 |
| Desktop | > 1024px | 三栏，顶部导航 |

### 7.3 无障碍规范

- 所有交互元素可键盘聚焦
- 颜色对比度 ≥ 4.5:1
- 按钮/链接最小点击区域 44×44px
- 进度状态需文字说明

## 八、模板选择器详细设计（重点）

### 8.1 模板卡片网格布局

```
桌面端 (≥1024px): 4列网格, 卡片间距 24px
平板端 (640-1024px): 3列网格, 卡片间距 16px
移动端 (<640px): 2列网格, 卡片间距 12px
```

### 8.2 模板预览交互流

```
1. 用户点击模板卡片
   ↓
2. 弹出 TemplatePreviewModal
   - 背景变暗 (overlay: rgba(0,0,0,0.6))
   - 模态框从中心放大入场 (scale 0.95→1, 250ms)
   ↓
3. 用户可左右滑动/点击箭头切换模板
   - 切换时：新图从右侧滑入，旧图向左侧滑出 (300ms)
   - 底部指示点同步更新
   ↓
4. 用户点击"使用此模板"
   - 模态框关闭 (scale 1→0.95, opacity 1→0, 200ms)
   - 模板自动选中，卡片显示选中态
   - 可选：自动滚动到编辑区
```

### 8.3 模板筛选交互

```
1. 点击筛选下拉框
   ↓
2. 下拉菜单从顶部向下展开 (max-height动画, 200ms)
   - 选项hover时背景变蓝 #DBEAFE
   - 选中项显示勾选图标
   ↓
3. 选择筛选条件后
   - 下拉框显示已选条件标签
   - 卡片网格重新渲染 (fade out → fade in, 150ms)
   - 显示"共 X 个模板"更新
   ↓
4. 清空筛选
   - 点击"X"标签或重置按钮
   - 所有卡片恢复显示
```
